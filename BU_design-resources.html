How to use this section?

This section is for staff only and should always be hidden from students.

This section contains placeholders and backup copies of important elements. Placeholders are pre-formatted elements that can be edited to incorporate your videos/textual information. Further instructions on how to replace the default text and videos with your own are provided within the edit screen of each element.

Instructions:

  1. Identify the element you would like to use
  2. Click: edit > duplicate
  3. Move the duplicated element to the page in which you want it displayed.
