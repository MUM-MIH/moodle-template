How to customise the task header:
The icons that appear on each task header can be changed easily by editing the icon names (e.g. 'fa-play-circle'). The text 'fa-3x' determines the size of the icon. Do not change this.

You will find a list of icon names here.
Replacing 'fa-play-circle' with 'fa-bell', for example, will change the 'play' icon to a 'bell' icon.

[fa-play-circle fa-3x fa-fw]
