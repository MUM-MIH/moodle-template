How to customise the weekly task list:
The icons that appear above each task description (e.g. 'watch videos') can be changed easily by editing the icon names (e.g. 'fa-play-circle'). 

You will find a list of icon names here.
Replacing 'fa-play-circle' with 'fa-bell', for example, will change the 'play' icon to a 'bell' icon.

Note: If you make changes to this table, you will also need to edit the corresponding task headings on this Moodle page.
