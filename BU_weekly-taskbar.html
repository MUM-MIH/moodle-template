How to customise the weekly taskbar:
The icons that appear above each task description (e.g. 'watch videos') can be changed easily by editing the icon names (e.g. 'fa-play-circle'). Replacing 'fa-play-circle' with 'fa-bell', for example, will change the 'play' icon to a 'bell' icon. 'fa-2x' sets the size of the icon - do not delete or change this.

You will find a list of icon names here.
Watch this video for step-by-step  instructions on how to edit and add anchor links.

Note: If you make changes to this table, you will also need to edit the corresponding task headings on this Moodle page.
