How to customise this header:
The icons that appear on each task header can be changed easily by editing the icon names (e.g. 'fa-play-circle'). Replacing 'fa-play-circle' with 'fa-bell', for example, will change the 'play' icon to a 'bell' icon. The text 'fa-3x' determines the size of the icon and 'fa-fw' aligns the icon. Do not change this. Do not change the font formats or colour. These will be styled automatically.

You will find a list of icon names here.
